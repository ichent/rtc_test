Pocket RussKey 2003 Lite 2.1
Компактный (Lite) русификатор для Windows Mobile 2003/2003 SE (установка с ПК или непосредственно на КПК)
MacCentre Pocket RussKey 2003 Lite предназначен для русификации карманных компьютеров, работающих под управлением Windows Mobile 2003/2003 SE в том случае, когда перевод интерфейса не требуется. 
При установке выберите режим "лайт-версия". 

ВНИМАНИЕ! На КПК с Pocket PC 2000/2002 используйте только русификатор Pocket RussKey Lite. 

MacCentre Pocket RussKey 2003 Lite в режиме "лайт-версия" обеспечивает полную поддержку русского языка на уровне ОС, при этом интерфейс (меню, иконки, системные сообщения и т.п.) остаётся английским, или на языке оригинальной ОС. Компактная (lite) версия работает практически со всеми карманными ПК платформ Windows Mobile 2003/2003 SE. 

Для распознавания перьевого ввода рекомендуется скачать и установить программу распознавания рукописи MacCentre Graphite. Ввод букв в этой программе аналогичен популярной системе ввода Граффити для Palm OS, но MacCentre Graphite можно настроить под индивидуальные особенности почерка, а также добавить новые наборы росчерков для различных языков. 

Основные возможности русификатора MacCentre Pocket RussKey 2003 (в режиме лайт-версии): 
- Поддержка российских форматов даты, времени и т.д. Сортировка с учётом русского алфавита. 
- Русская экранная клавиатура, с поддержкой различных режимов ввода. 
- Базовые кодировки русского языка, что позволяет работать в Рунете. 
- Широкий спектр совместимости русификатора с различными моделями Windows Mobile 2003/2003 SE. 
ВНИМАНИЕ! Pocket RussKey 2003 не предназначен для устаревших компьютеров типа Palm-Size PC и Palm PC. 

Полная (Full) версия MacCentre Pocket RussKey 2003, кроме функциональности лайт-версии, добавляет перевод интерфейса на русский язык, поддержку аппаратных клавиатур и другие модули. Перевод интерфейса выполнен отключаемым, начиная с версии Pocket RussKey 2003 2.00 имеется возможность установки дополнительных языковых модулей (если таковые есть). 
MacCentre Pocket RussKey 2003 совместим с различными КПК платформ Pocket PC 2002, Windows Mobile 2003 и Windows Mobile 2003 SE. 

Информация о полной версии MacCentre Pocket RussKey 2003 находится на странице продукта.
